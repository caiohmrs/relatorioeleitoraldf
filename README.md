# relatorioeleitoraldf

#Este app é uma previa de um script para gerar relatorios profundos sobre cada candidato do DF nas eleições de 2022, separando por local de votação e trazendo insights previos sobre cada candidato frente a eleição que concorreu. Na prévia, apenas o relatório para governadores está disponivel, porém caso deseje uso total do aplicativo entrar em contato comigo.
